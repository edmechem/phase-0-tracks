Directions for loading Google fonts to webpage.

  1. Go to the Google Fonts page.
  2. When you find a font that you like, click the Quick-Use button (it's the right-pointing arrow).
  3. On the new page that you are taken to, select the style(s) that you want.
  4. Select your character set (default is Latin).
  5. Copy the href code and paste it up into the head of your html.
  6. Copy the font-family code and paste into your CSS or as an inline style.
  7. Now check out your fancy fonts on your website!
